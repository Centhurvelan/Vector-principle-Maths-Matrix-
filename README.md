## Requirements

- Python 3.8+ recommended
- Jupyter (notebook or lab)
- numpy
- matplotlib (optional, for plots)

## Installation

1. Clone the repository:

   git clone https://github.com/Centhurvelan/Vector-principle-Maths-Matrix-.git
   cd Vector-principle-Maths-Matrix-

2. (Optional) Create and activate a virtual environment:

   python -m venv venv
   source venv/bin/activate   # macOS / Linux
   venv\Scripts\activate    # Windows

3. Install dependencies:

   pip install -r requirements.txt

If a requirements.txt is not present, install the main packages manually:

   pip install jupyter numpy matplotlib

## Usage

- Launch Jupyter Notebook from the repository root:

  jupyter notebook

- Open the notebook file (assignment.ipynb) and run the cells to reproduce the results.
